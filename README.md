# Toxic Comment Classification System

## 📌 Description
This project is a machine learning-based system designed to classify text comments as toxic or non-toxic using Natural Language Processing (NLP) techniques.

## 🚀 Features
- Text preprocessing and cleaning
- Feature extraction using TF-IDF
- Toxic vs Non-toxic classification
- Model evaluation using metrics and confusion matrix
- Interactive web interface using Gradio

## 🛠 Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy
- Matplotlib
- WordCloud
- Gradio

## ⚙️ Model
The system uses TF-IDF vectorization for text representation and Logistic Regression for classification. It processes user input text and predicts whether the comment is toxic or not.

## 📊 Workflow
- Data loading and preprocessing
- Text vectorization using TF-IDF
- Model training using Logistic Regression
- Evaluation using classification report and confusion matrix
- Deployment using Gradio interface
